## How to Contribute Translations

### Become a Language Lead

**Requirements**:
- Native or near-native fluency
- Technical background (Git/GitHub)
- Available for coordination and reviews
- Commitment to quality

**Responsibilities**:
- Coordinate language-specific translations
- Review and approve translations
- Maintain glossary for your language
- Support other translators

**Benefits**:
- Recognition as language lead
- Free Pro subscription
- Early access to features
- Direct communication with core team

### Join a Translation Team

**Steps**:
1. Review [Translation Guide](TRANSLATION_GUIDE.md)
2. Join [Translation Discord](https://discord.gg/rafgittools)
3. Introduce yourself in #translations
4. Request assignment to language team
5. Start with small document
6. Grow to larger contributions

### Start a New Language

**Requirements**:
- Language not yet supported or planned
- Commitment to translate priority documents
- Native speaker or certified translator
- Technical expertise

**Process**:
1. Propose language addition
2. Get approval from documentation team
3. Set up language directory structure
4. Create glossary for technical terms
5. Begin with README translation
6. Build translation team
